# Smart Bookmark App

A simple bookmark manager that allows users to sign in with Google, add private bookmarks, and see updates in real-time. Styled with Tailwind CSS and deployed on Vercel.

## Live App
[https://smart-bookmark-app.vercel.app](https://smart-bookmark-app.vercel.app)

## Features
- Google OAuth login (no email/password)
- Add bookmarks with **Title** and **URL**
- Bookmarks are private per user
- Real-time updates: bookmarks added in one tab appear in another instantly
- Delete your own bookmarks
- Clean responsive design with Tailwind CSS

## Tech Stack
- **Next.js** (App Router)
- **Supabase** (Authentication, Database, Realtime)
- **Tailwind CSS**
- **Vercel** (for deployment)
